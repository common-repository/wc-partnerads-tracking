== Description ==

Add the required PartnerAds tracking code to the Woocommerce installation in the thankyou page.
Signup at http://www.partner-ads.com/ to get your campaignID for this plugin.

== Installation ==

Upload the WC PartnerAds Tracking to your wordpress, Activate it.

1, 2, 3: You're done!
